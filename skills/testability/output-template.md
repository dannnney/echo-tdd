# 可观测性方案输出模板

阶段一的输出是一份**可观测性方案** —— 回答"我们能通过什么通道触发系统、通过什么通道观测结果"。

可观测性方案是后续所有阶段的基础：阶段二验证通道可用性，阶段三基于通道设计用例。如果可观测性没梳理清楚，后续一切都会跑偏。

根据实际情况填充，不适用的部分可以省略。
