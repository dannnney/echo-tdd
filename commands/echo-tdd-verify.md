description: "Verify an Echo-TDD plan against the current environment and record executable channels"
---

Use the `echo-tdd-verify` skill.

This command should:

- load the relevant Echo-TDD plan
- probe the environment and validate channel availability
- record the outcome as a verification report
- generate minimal helper or scaffold code when needed to prove the plan is executable

Preferred artifact path: `docs/echo-tdd/<topic>/verify.md`

Primary slash command: `/echo-tdd:verify`
